# Contributing

MIT. No CLA.

- Run `uv sync && pytest` before opening a PR.
- New corpus clips welcome; see [`docs/corpus.md`](docs/corpus.md).
- Prompt changes go in `mouthflow/prompts/`, not code.

## Code of conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
Enforcement questions: open an issue or email the maintainer.
